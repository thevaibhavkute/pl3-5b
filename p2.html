Sum & Average Program

Problem Statement:

Write a program in any language to read 10 numbers from keyboard and find their sum and average.

Code:

            
                //C++ code

                int main() 
                {

                    int sum = 0;
                    float average = 0;

                    int numbers[10];

                    for(int i = 0; i < 10; i++){
                        cout << "Enter Number " << i+1 << ": ";
                        cin >> numbers[i];
                    }

                    for(int i = 0; i < 10; i++){
                        sum += numbers[i];
                    }

                    average = sum/10;

                    cout << "\nThe sum of the numbers is " << sum << endl;
                    cout << "\nThe average of the numbers is " << average << endl;

                    return 0;
                }

Sample Input:

            
                Enter number 1: 10
                Enter number 2: 20
                Enter number 3: 30
                Enter number 4: 40
                Enter number 5: 50
                Enter number 6: 60
                Enter number 7: 70
                Enter number 8: 80
                Enter number 9: 90
                Enter number 10: 100

Sample Output: 

            
                The sum of the numbers is 550
                The average of the numbers is 55

Variables used in the program:

sum, i,average

Complexity:

Time:

O(n)

Space:

O(n)
